## Обектно-ориентирано програмиране. Структури от данни (записи).

### Обектно-ориентирано програмиране

#### Дефиниция
- Процедурно програмиране
- ООП: Парадигма в компютърното програмиране, при която една програмна система се моделира като набор от обекти, които взаимодействат помежду си.


#### Основни ООП принципи
- Абстракция
- Капсулация
- Наследяване
- Полиморфизъм

### Структури от данни (записи)

#### Дефиниция
```
struct <име на структурата> {   
  <полета>                              
  <методи>                              
};
```

- **поле (член-данна)**: променлива от даден тип, принадлежаща към данните на структурата <br>
- **метод (член-функция)**: функция, вътрешна за дадена структура

#### Употреба
```
    <име на структурата> <име на променливата>;
    <име на променливата>.<име на поле>;
    <име на променливата>.<име на метод>(<параметри>);
```

### Модификатори за достъп

| Модификатор | Достъп от външния свят | Достъп от наследниците | Достъп от вътрешните методи |
| ----------- | ---------------------- | ---------------------- | --------------------------- |
| public      | Да                     | Да                     | Да                          |
| protected   | Не                     | Да                     | Да                          |
| private     | Не                     | Не                     | Да                          |

```
struct <име на структурата> {
private:
    <скрити полета>                              
    <скрити методи>  
public:
    <публични полета>                              
    <публични методи>                              
};
```
