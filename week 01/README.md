### Структури от данни (записи)

#### Дефиниция
```
struct <име на структурата> {   
  <полета>                              
  <методи>                              
};
```

- **поле (член-данна)**: променлива от даден тип, принадлежаща към данните на структурата <br>
- **метод (член-функция)**: функция, вътрешна за дадена структура

#### Употреба
```
    <име на структурата> <име на променливата>;
    <име на променливата>.<име на поле>;
    <име на променливата>.<име на метод>(<параметри>);
```

### Нива на достъп

- **public:** полетата и методите са достъпни извън дефиницията на структурата
  - по подразбиране, всички полета и методи в дадена структура са публични
- **private:** полетата и методите са достъпни само в дефиницията на структурата
- **protected:** TBD

```
struct <име на структурата> {
private:
    <скрити полета>                              
    <скрити методи>  
public:
    <публични полета>                              
    <публични методи>                              
};
```
