## Класове. Конструктори

#### Дефиниция

```
class <име на класа> {   
  <полета>                              
  <методи>                              
};
```

#### Разлики между структура и клас

- Модификатор на достъп по подразбиране
  - Public за структурите
  - Private за класовете
- Употреба
  - Структури - малки пакети от данни
  - Класове - тежка логика
